Ejecución del servidor

Cuando se ejecuta el comando node server.js, el servidor inicia y queda escuchando solicitudes en el puerto 3000.

En la terminal aparece el mensaje:
Servidor ejecutándose en puerto 3000

Cuando se visita la ruta principal /, el navegador muestra:
Servidor activo y funcionando

Cuando se visita la ruta /estado, el servidor responde con un objeto JSON:

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}

Esto demuestra que el servidor está activo y puede responder diferentes solicitudes.
