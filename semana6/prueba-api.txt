Prueba de la ruta POST /registro

Para probar la ruta /registro se envía una petición POST con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor recibe los datos enviados en req.body y responde con un JSON organizado.

Respuesta esperada:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
